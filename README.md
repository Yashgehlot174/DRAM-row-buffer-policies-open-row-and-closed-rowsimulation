## ğŸ§  DRAM Timing Concepts

### ğŸ”¹ Row Buffer Hit  
Requested row is already open â†’ **CAS** latency.

### ğŸ”¹ Row Buffer Miss  
Row is closed â†’ **RAS + CAS** latency.

### ğŸ”¹ Row Buffer Conflict  
Another row is open â†’ **PRE + RAS + CAS** latency.

The simulator computes internal timing components:

```
CAS = ROWBUF_HIT_LATENCY  
RAS = ROWBUF_MISS_LATENCY - CAS  
PRE = ROWBUF_CONFLICT_LATENCY - (RAS + CAS)
```
